# 房源多渠道营销文案撰写流程

此工作流用于为已经定好价格的房源撰写极具煽动性和吸引力的多平台营销文案。

1. **提炼核心卖点 (Extract Core Selling Points)**
   - 基于房产类型找出最吸引人的3个特点：
     - 如果是楼花转让：“0 转让费”、“平转/亏转捡漏”、“开发商信誉”。
     - 如果是现房/二手房：“交钥匙拎包入住 (Move-in Ready)”、“近期翻新 (Upgraded)”、“无管理费 (Freehold)”或“顶级学区”。

2. **撰写英文版 Listing 文案 (English Kijiji/Marketplace Copy)**
   - **标题**：必须使用 Emoji 且突出最大卖点和优惠（如 "🔥 MUST SELL / 🎈 Move-in Ready"）。
   - **正文**：强调 FSBO (For Sale By Owner) 的优势，列清价格、房屋参数、附属设施（车位/Lot Size等）。
   - **结尾**：注明欢迎带客经纪 (Co-op commission considered) 和联系方式。

3. **撰写中文版社交媒体文案 (Chinese Social Media Copy for 小红书/约克论坛)**
   - **标题**：采用“捡漏”、“房东直售”、“宝藏房源”等具有社区传播属性的词汇。
   - **正文**：使用接地气的语言，突出痛点（如“省去高额转让费”或“免去双边经纪费直给底价”）。适当配合分段和 Emoji。
   - **标签**：添加相关标签（如 #多伦多买房 #多伦多楼花转让 #多伦多二手房 #房东直售 等）。

4. **发帖注意事项与渠道建议 (Posting Guidelines & Channels)**
   - 列出推荐的 Facebook Groups 和本地论坛。
   - 提醒合规事项：针对楼花，不得泄露原合同照片；针对现房，建议提供优质实拍图或虚拟 staging 图片。
