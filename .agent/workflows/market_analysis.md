# 房产市场与竞品对标分析流程

此工作流用于在接手一个新的房源（现房或楼花转让）时，进行全面的市场背景调查和定价分析。

1. **多维度信息收集与宏观分析 (Multi-dimensional Data Gathering & Macro Analysis)**
   - **房产基本参数**: 确定目标房产的具体位置、类型（楼花转让 Assignment 或 现房 Resale；Condo 或 House）、户型等。
   - **房屋买卖市场态势**: 确定当前市场的宏观状态（时间节点、买方/卖方/平衡市场、平均挂牌天数 DOM、库存量）。
   - **租房市场动向 (Rental Market)**: 调研周边区域的**房租上涨/下降趋势**、平均租出天数、投资客比例及整体租售比状况。
   - **宏观环境与区域就业 (Macro & Employment)**: 收集目标地的**就业率/失业率**趋势、核心产业发展情况，并结合大环境（通胀、经济周期）评估对房产价格的支撑力。

2. **竞品检索 (Competitor Search)**
   - 搜索或模拟查询同一区域类似房型的近期挂牌价（Active Listings）和成交价（Sold Comparables）。
   - 重点关注包含类似稀缺资源（如双车库、全新装修、无敌视野）的竞品。

3. **定价策略评估 (Valuation & Pricing Strategy)**
   - 根据卖方的核心诉求（如：急需套现、追求最大利润等），制定不同的定价方案。
   - **楼花转让**：分析“原价平转”、“亏损转让”或“溢价转让”在当前市场中的成功率。
   - **现房出售**：分析“低价钓鱼（Bidding War）”策略与“合理市价（Market Value）”策略的利弊。

4. **输出核心分析报告 (Generate Report)**
   - 将分析结果整理为多维度的专业报告，包含：
      1. 区域基本面概览（涵盖宏观环境、就业率、租房市场及租金趋势）。
      2. 房屋买卖市场走势及当前均价概述。
      3. 核心竞品对比（挂牌中与已售出对比，必要时加入租售比回报对比）。
      4. 推荐的“挂牌叫价 (Asking Price)”与灵活的底线价策略。
